Интеллектуальная информационная система (экспертная система) для предварительной диагностики заболеваний на основе введенных симптомов. Программа демонстрирует работу алгоритмов прямого и обратного вывода, а также взаимодействие с базами данных SQL.

Основные возможности:
Прямой вывод (Forward Chaining): Определение наиболее вероятного диагноза на основе списка симптомов, введенных пользователем.

Обратный вывод (Backward Chaining): Поиск необходимых симптомов и рекомендаций по названию конкретной болезни.

Механизм объяснения: Система не просто выдает результат, но и объясняет, на основании каких данных был поставлен диагноз.

Постоянство данных: Использование базы данных SQLite для хранения правил, симптомов и текущего состояния сессии.

Графический интерфейс: Удобное окно на Tkinter для управления списком симптомов.

Технологический стек:
Язык: Python 3.x

Библиотека GUI: Tkinter (стандартная библиотека)

База данных: SQLite3

Архитектура: Использование паттерна контекстных менеджеров (closing) для безопасной работы с БД.

Как запустить проект:
Клонируйте репозиторий:

Bash
git clone https://github.com/CheburashkaZ/PythonProject.git
cd PythonProject
Запустите программу:
Никаких дополнительных библиотек устанавливать не нужно (используются только встроенные модули Python).

Bash
python main.py
Как пользоваться:
Добавление данных: Введите симптом (например, "кашель") в текстовое поле и нажмите "Добавить".

Диагностика: После ввода нескольких симптомов нажмите "Диагностика". Система сопоставит их с базой правил и выдаст результат с рекомендациями.

Поиск по болезни: Если вы хотите узнать симптомы конкретной болезни (например, "солнечный удар"), введите название в нижнее поле поиска.

Структура базы данных:
При первом запуске программа автоматически создает файл expert_system.db со следующими таблицами:

rules — названия диагнозов.

rule_symptoms — связи между болезнями и их симптомами.

rule_recommendations — список действий для лечения.

current_symptoms — сохранение ваших введенных данных между запусками.
